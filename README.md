## Sistema CRUD em Python 

Projeto de estudo desenvolvido em Python, com foco em lógica de programação, POO, persistência de dados e boas práticas iniciais de segurança.

Este sistema roda no terminal e implementa as operações básicas de um CRUD, incluindo login de usuário e armazenamento em arquivo JSON.

## Funcionalidades

Cadastro de usuários

Login com email e senha

Senhas armazenadas de forma segura (hash)

Validação simples de email

Listagem de usuários

Atualização de dados

Remoção de usuários

Persistência dos dados em arquivo JSON

## Tecnologias Utilizadas

Python 3

Módulos padrão:

json

os

re

hashlib

Nenhuma biblioteca externa é necessária.
